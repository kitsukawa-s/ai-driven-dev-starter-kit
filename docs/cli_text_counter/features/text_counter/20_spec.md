# 機能名

text_counter

## 作りたいもの

入力文字列を受け取り、文字数を整数として返すシンプルなCLI向け機能を作ります。

`text_counter` feature は文字数の算出だけを担当します。
CLI引数の受け取り、標準出力への表示、終了コードの返却は entrypoint の責務として扱います。

## 実行イメージ

```bash
python src/cli_text_counter/entrypoint.py --text "hello"
```

## 期待する出力

```text
5
```

## 必須要件

- 入力文字列の文字数を数える
- 文字数は Python の `len()` 相当で数える
- 空文字列は許可し、文字数 `0` を返す
- feature は文字数を整数として返す
- entrypoint は feature が返した整数を標準出力に表示する
- CLI引数が不足している場合は entrypoint 側でCLI引数エラーにする
- 外部ライブラリは使わない
- 初学者が読めるシンプルな実装にする

## 入力

- feature は文字列を受け取る
- entrypoint はCLI引数で入力文字列を受け取る
- 空文字列は有効な入力として扱う

## 出力

- feature は入力文字列の文字数を表す整数を返す
- entrypoint は feature が返した整数を標準出力に表示する

## エラー扱い

- 入力文字列が指定されていない場合は、entrypoint 側で `argparse` の引数エラーにする
- 空文字列が指定された場合はエラーにしない
