# 機能名

simple-calculator

## 作りたいもの

2つの整数を受け取り、足し算の結果を表示するCLIツールを作ります。

## 実行イメージ

```bash
python src/cli_simple_calculator/entrypoint.py --a 2 --b 3
```

## 期待する出力

```text
5
```

## 必須要件

- `--a` で1つ目の整数を指定できる
- `--b` で2つ目の整数を指定できる
- 2つの整数を足し算した結果を標準出力に表示する
- 引数が不足している場合はCLI引数エラーにする
- 外部ライブラリは使わない
- 初学者が読めるシンプルな実装にする

## 入力

- CLI引数 `--a`
- CLI引数 `--b`
- どちらも整数

## 出力

- 足し算の結果を標準出力に表示する

## エラー扱い

- `--a` または `--b` が不足している場合は、`argparse` の引数エラーにする
- 整数として解釈できない値は、`argparse` の型変換エラーにする
