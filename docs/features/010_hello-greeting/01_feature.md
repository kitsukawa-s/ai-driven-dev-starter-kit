# 機能名

hello-greeting

## 作りたいもの

入力された名前に対して、あいさつ文を返すCLIツールを作ります。

## 実行イメージ

```bash
python src/hello_greeting.py --name Alice
```

## 期待する出力

```text
Hello, Alice!
```

## 必須要件

- `--name` で名前を指定できる
- 名前が指定された場合、`Hello, <name>!` を標準出力に表示する
- 名前が指定されない場合、CLI引数エラーにする
- 空文字はエラー扱いにする
- 空白のみの文字列はエラー扱いにする
- 外部ライブラリは使わない
- 初学者が読めるシンプルな実装にする

## 入力

- CLI引数 `--name`
- 値は文字列

## 出力

- 標準出力に `Hello, <name>!` を表示する
- `main` は正常終了時に `0` を返す

## エラー扱い

- `--name` が指定されない場合は、`argparse` の引数エラーにする
- 空文字、空白のみの文字列は `ValueError` にする
