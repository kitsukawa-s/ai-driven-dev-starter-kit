# 関数呼び出し定義を作成してください

あなたは、このリポジトリのAI駆動開発ルールに従う開発補助者です。

利用者は、下の「利用者が指定する項目」だけを主に書き換えて使います。
それ以外の作業ルールは変更せず、`AGENTS.md` と `docs/templates/` に従ってください。

## 利用者が指定する項目

- 対象機能フォルダ: `docs/features/<number>_<feature-name>/`
- 対象機能名: `<feature-name>`
- 作りたいもの: `必要に応じて補足を書く`
- 補足条件: `必要に応じて書く。なければ「なし」`

## 参照するファイル

- `AGENTS.md`
- `<対象機能フォルダ>/01_feature.md`
- `<対象機能フォルダ>/02_function_design.md`
- `docs/templates/03_function_call_flow_template.md`

## 作成または更新するファイル

- `<対象機能フォルダ>/03_function_call_flow.md`

## 作業範囲

1. 仕様と関数設計を確認してください
2. 関数設計に矛盾や過剰な責務がないか簡潔にレビューしてください
3. `docs/templates/03_function_call_flow_template.md` の見出し構成を維持して `03_function_call_flow.md` を作成または更新してください

## 禁止事項

- まだ実装しないでください
- テスト設計、テスト作成へ勝手に進まないでください
- CLI処理とロジック処理の責務を混ぜないでください
- 仕様にない処理を追加しないでください
- 共通化候補があっても `src/common/` へ切り出さないでください
